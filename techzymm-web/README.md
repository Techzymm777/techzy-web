# Techzy (Static Site)

Static multi-page website (HTML/CSS/JS) for a minimalist laptop + desktop shop.

## Pages

- `index.html` (Home)
- `about.html` (About Us)
- `products.html` (Products with cards + search/filter/sort)
- `contact.html` (Contact form that composes an email via `mailto:`)

## Run locally

You can open `index.html` directly, or serve the folder:

```bash
cd monochrome-computers
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
